Comandi di Vim essenziali

| comando | descrizione |
|:----------|:-------|
| `:set mouse=a`| abilita l'utilizzo del mouse |
| `:w` | salva il file |
| `:q` | uscita |
| `:x` | salva e uscita |
| `:a` | applica il comando a tutte le tab di vim aperte |
| `:E` | esplora tra i file |
| `:Ve` | apri verticalmente una tab di esplora |
